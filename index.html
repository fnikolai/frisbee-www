Install in seconds.

No dependencies, No compilation.

Install via Helm

>> helm repo add frisbee https://carv-ics-forth.github.io/frisbee/charts
>> helm upgrade --install --wait my-frisbee frisbee/platform

See the Getting Started Guide for other methods.
